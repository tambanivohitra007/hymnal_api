API Endpoints

Use these URLs to access the data programmatically:

  • GET api/hymns.json — All hymns (no lyrics)
  • GET api/hymns/{id}.json — Single hymn with lyrics
  • GET api/categories.json — All categories
  • GET api/categories/{id}.json — Hymns by category
